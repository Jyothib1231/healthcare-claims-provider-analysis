# Healthcare Claims & Provider Performance Analysis

This project analyzes healthcare claims data to identify approval trends, denial patterns, and provider performance using Python and SQL.

## Tools
- Python (Pandas, Matplotlib, Seaborn)
- SQL
- Healthcare analytics KPIs

## Key Insights
- Identified claim approval trends
- Evaluated provider performance
- Analyzed processing times and denial drivers

This project demonstrates real-world data analyst skills aligned with healthcare analytics roles.
